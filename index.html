Paket.Unity3D

Paket.Unity3D

An extension for the Paket dependency manager that enables the integration of NuGet dependencies into Unity3D projects.

Features

  • NuGet dependencies
  • Dependencies as DLL or source-files
  • Plugin & native code support
  • based on Paket

How to get Paket.Unity3D

Download the latest Paket.Unity3D executable from:

  • GitHub

Prerequisites

Windows

You’ll need .NET Framework 4.0

Unix/Linux

You’ll need Mono MRE 4.0.2
